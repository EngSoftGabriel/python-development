# 5. Dashboard Simples

Dashboard com 4 gráficos usando `matplotlib`:

- Linha — vendas mensais
- Barras — vendas por categoria
- Pizza — participação por categoria
- Dispersão — investimento × retorno

## Como executar

```bash
pip install -r requirements.txt
python main.py
```

A imagem `dashboard.png` é gerada e exibida na tela.
