# 8. Organizador de Arquivos

Organiza arquivos automaticamente em subpastas por categoria
(Imagens, Documentos, Vídeos, Áudios, Compactados, Códigos,
Executáveis, Outros).

## Como executar

```bash
# Organiza a pasta Downloads (padrão)
python main.py

# Organiza uma pasta específica
python main.py "C:\caminho\para\pasta"

# Simula sem mover (recomendado na primeira execução)
python main.py --dry-run
```

Sem dependências externas — usa apenas a biblioteca padrão.
