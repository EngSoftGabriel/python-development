# 3. Automação de Planilha

Gera uma planilha Excel de vendas usando `openpyxl`, com:

- Cabeçalho estilizado
- Fórmulas automáticas (Total e Total Geral)
- Formatação monetária e larguras ajustadas

## Como executar

```bash
pip install -r requirements.txt
python main.py
```

O arquivo `relatorio_vendas.xlsx` será criado nesta pasta.
